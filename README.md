# 🚀 Habit Tracker

A modern and responsive habit-tracking application built with **React**, **TypeScript**, and **Tailwind CSS**. Track your daily habits, maintain streaks, and stay consistent with a clean weekly calendar interface.

## 📸 Overview

Habit Tracker helps users build positive habits by providing an easy way to:

* Create and manage daily habits
* Mark habits as completed
* Track progress across weeks
* Monitor consistency through streaks
* Save data automatically using browser local storage

## ✨ Features

### ✅ Habit Management

* Add new habits instantly
* Delete habits when no longer needed
* Manage multiple habits simultaneously

### 📅 Weekly Calendar View

* Navigate between weeks
* View habit completion history
* Track progress day by day

### 🔥 Streak Tracking

* Automatically calculates habit streaks
* Encourages consistency and accountability

### 💾 Local Data Persistence

* Uses browser Local Storage
* Data remains available after page refresh or browser restart

### 📱 Responsive Design

* Works smoothly on desktop, tablet, and mobile devices
* Clean and modern user interface

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
https://github.com/Sabbir12341/Personal-Habit-Tracker.git
cd habit-tracker
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Development Server

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```

## 🎯 How to Use

1. Enter a habit name and click **Add Habit**
2. Click on a day to mark the habit as completed
3. Navigate between weeks using the week controls
4. View your current streak count
5. Delete habits when necessary
